## Guide Template

When creating a new guide, use this structure:

```markdown
# [Guide Title]

## Overview
Brief description of what this guide covers.

## Prerequisites
- Required knowledge
- Required setup
- Required permissions

## Steps

### Step 1: [Step Title]
Description of what to do.

**Example:**
\`\`\`bash
# Code example
\`\`\`

### Step 2: [Step Title]
Continue with next steps...

## Common Issues

### Issue 1
**Problem:** Description
**Solution:** How to fix it

## See Also
- Related guides
- Related API documentation
```
